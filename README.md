# Projet Zookeeper - Hyperskill 

Ce projet est conçu pour les développeurs débutants en kotlin sur la plateforme JetBrains Academy (Hyperskill). Il permet d'assimiler les concepts fondamentaux de la programmation en créant un outil interactif en ligne de commande.

## À propos

L'objectif principal de ce projet est de comprendre les bases de la syntaxe et d'apprendre à travailler avec :
* Les variables et leur déclaration.
* Les types de stockage de données, tels que les listes ou les chaînes de caractères complexes.
* Les structures de contrôle, notamment les boucles interactives (`while`) et les conditions d'affichage (`when`).

## Objectifs d'apprentissage

En complétant les étapes de ce projet, les concepts suivants sont mis en pratique :
* Récupérer et traiter de manière sécurisée les entrées de l'utilisateur depuis la console.
* Créer une boucle d'exécution continue qui reste active jusqu'à ce qu'une commande de sortie explicite (comme le mot "exit") soit saisie.
* Organiser et formater correctement de grands blocs de texte multilingnes.

## Technologies utilisées

* **Langage :** Kotlin
* **Environnement de développement :** IntelliJ IDEA
* **Gestionnaire de dépendances :** Gradle

## Comment exécuter le projet

1. Ouvrez le projet dans IntelliJ IDEA.
2. Assurez-vous que les paramètres Gradle pointent vers la version du JDK requise (ex: Java 17 ou Java 23 selon la configuration de votre machine).
3. Attendez la fin de la synchronisation Gradle.
4. Lancez l'exécution du fichier principal (ex: `ZookeeperKt`).
5. Suivez les instructions affichées dans le terminal pour interagir avec le programme et afficher les différents habitats.
